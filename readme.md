## 說明

這是一個使用 Express 框架建立的單純 RESTful API數據庫，並且使用 Swagger 產出 API 文件。

### 啟動步驟：

1. 複製資料庫：

   在終端機中使用以下命令來複製資料庫：

   ```bash
   git clone https://github.com/wwwchendev/restfulAPI.git
   ```

2. 進入複製的目錄：

   ```bash
   cd restfulAPI
   ```

3. 安裝 npm 套件：

   執行以下命令以安裝所需的 npm 套件：

   ```bash
   npm install
   ```

4. 執行伺服器：

   運行以下命令以啟動伺服器：

   ```bash
   node server.js
   ```

5. 開啟 API 文件：

   在瀏覽器中打開以下連結：

   [http://localhost:5000/api-docs/](http://localhost:5000/api-docs/)

   這將會開啟 API 文件，讓你可以查看 API 的詳細資訊。
